# 每日古画 (DailyArt) - 极简中国传统艺术推送 PWA

## 项目简介
这是一个专注于中国传统古画艺术鉴赏的渐进式 Web 应用 (Progressive Web App, PWA)。它采用了“单页沉浸式”交互设计，致力于每日推送一幅高分辨率的传统艺术作品及其历史背景解析。

本项目的核心目标是**探索轻量级前端架构在跨终端适配与断网环境下的表现力**，以最低的计算成本提供近似原生 App 的阅读体验。

## 核心特性
* **沉浸式 UI 设计**：基于 `CSS Viewport` 单位与 `backdrop-filter` 毛玻璃效果，实现无边距全屏适配与内容悬浮交互。
* **本地数据注入引擎**：摒弃传统后端请求，利用原生 JavaScript 构建前置数据索引逻辑，基于客户端系统日期实现内容的毫秒级切换。
* **PWA 断网缓存支持**：完整配置了 `manifest.json` 与 `Service Worker`，拦截并托管网络请求（采用 Network First 策略），实现在无网/弱网环境下的“0 延迟”秒开。
* **全端响应式部署**：可作为传统网页在任何现代浏览器中访问，亦可通过浏览器级菜单“安装”为手机桌面或 PC 桌面的独立应用形态。

## 技术栈与选型理由
* **HTML5 / CSS3**：不依赖任何第三方 UI 库，纯手写 DOM 结构与基础样式，保证页面体积极致轻量（核心代码 < 500 行）。
* **Vanilla JavaScript (原生 JS)**：避免引入 React/Vue 等框架增加学习成本和打包负担，纯原生编写事件监听与 DOM 渲染逻辑。
* **Vibecoding (AI 辅助开发)**：在代码生成环节结合了 Cursor + 大模型工作流，提升开发效能。
* **部署环境**：依托 GitHub Pages 自动化静态托管服务构建。
